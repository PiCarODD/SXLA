# SXLA
sqli+xss+lfi+admin-finder all in one script
# Usage
<ul>
  For SQLI
  <li>python sxla.py -s terget_website></li>
   For XSS
   <li>python sxla.py -x target_website</li>
   For LFI
    <li>python sxla.py -l target_website</li>
   For Admin Finder
     <li>python sxla.py -a target_website</li>
       </ul>
