# SXLA
sqli+xss+lfi+admin-finder all in one script
# Useage
<ul>
  For SQLI
  <li>python sxla.py -s <terget website></li>
   For XSS
   <li>python sxla.py -x <target website></li>
   For LFI
    <li>python sxla.py -l <target website></li>
   For Admin Finder
     <li>python sxla.py -a <target website></li>
       </ul>
